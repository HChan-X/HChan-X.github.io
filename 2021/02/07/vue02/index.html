Vue学习笔记02

发表于2021-02-07|更新于2021-02-07|笔记Vue
|阅读量:

Vue.js常见的使用方式有：

  • 直接引入，通过script标签引入完整的vue文件
  • webpack自定义打包
  • vue-cli脚手架，也是通过webpack进行打包

Vue 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件

响应的数据绑定：就是vue.js会自动响应数据的变化情况，并且根据用户在代码中预先写好的绑定关系，对所有绑定在一起的数据和视图内容都进行修改
